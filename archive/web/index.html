CoAI Gate 3 — Physical Units (Dimensional Consistency)

Gate 3 checks that each equation is dimensionally meaningful (e.g., prevents J ≤ bit). It does not prove a claim is true; it proves the units are consistent and obligations are checkable.

1) Select Mode

2) Enter Axioms (S-expressions)

One axiom per line. Supported forms include forall, =, <=, implies, and, or, not.
